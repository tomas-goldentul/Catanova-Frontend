# 📱 Catanova - Frontend
> **"Tu negocio, en orden."** — Panel de administración inteligente diseñado para emprendedores del rubro textil.

Catanova es la solución digital para quienes venden por WhatsApp e Instagram. No es una tienda online para clientes, sino una herramienta interna para reemplazar el cuaderno y el Excel por un dashboard profesional.

## 🚀 Funcionalidades Principales
* **📊 Dashboard de Control:** Visualización de ingresos, gastos y salud financiera del negocio.
* **📦 Gestión de Inventario:** Registro y control de stock optimizado para el rubro textil.
* **🛒 Registro de Ventas:** Interfaz rápida para cargar pedidos acordados manualmente.
* **📄 Catálogo Dinámico:** Generación de catálogos PDF/Imagen actualizados para compartir por WhatsApp.
* **🤖 Interfaz de Asistente IA:** Chatbot integrado que interpreta los datos del negocio para dar consejos estratégicos.

## 🛠️ Stack Tecnológico
* **Framework:** React.js
* **Lenguaje:** JavaScript
* **Estilos:** Tailwind CSS
* **Gestión de Estado:** Context API / Hooks
